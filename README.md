# Osama Elbably - Personal Portfolio

A modern, high-performance personal portfolio built for **Osama Elbably**, a Full-Stack Web Developer and AI Engineer. The design features a smooth **Ocean Blue & Cyan** color scheme, inspired by nature, combined with state-of-the-art UI elements.

## ✨ Key Features

- **Glassmorphic Floating Sidebar**: A unique, fully custom navigation pill that floats elegantly on desktop and transforms into a macOS-style dock on mobile devices.
- **Ocean Blue Theme**: A visually relaxing and professional palette (`#00A8FF` and `#00E5FF`).
- **Dynamic Projects Showcase**: High-quality visuals for Web Development, AI, and Data Science projects.
- **Interactive UI**: Custom mouse glows, digital particle effects, and animated background orbs.
- **Fully Responsive**: Adapts flawlessly to desktops, tablets, and smartphones.
